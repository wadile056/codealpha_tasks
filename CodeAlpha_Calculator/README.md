# CALC — Modern Calculator

A sleek, production-grade calculator built with HTML, CSS, and JavaScript.

## Features

- **All arithmetic operations**: Addition (+), Subtraction (−), Multiplication (×), Division (÷)
- **Special functions**: Percentage (%), Toggle sign (+/−), All Clear (AC)
- **Real-time display** with expression history
- **Keyboard support** — full keyboard input:
  - Numbers: `0–9`
  - Operators: `+`, `-`, `*`, `/`
  - Equals: `Enter` or `=`
  - Decimal: `.` or `,`
  - Clear: `Escape` or `Delete`
  - Backspace: `Backspace`
  - Percent: `%`
- **Responsive design** — works on mobile and desktop
- **Micro-interactions**: ripple effects, button animations, error shake

## Design

Industrial bold dark theme with:
- Gold operator buttons
- Red equals button
- Dot-grid background
- Space Mono monospace font
- Syne display font

## Usage

Simply open `index.html` in any modern browser. No build tools or dependencies required (fonts load from Google Fonts).
